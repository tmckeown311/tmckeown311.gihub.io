Chess Program

Chess is a common game that has many complex rules. This project appeared out of a desire to practice coding using inheritance and looking to solve specific niche problems.

This project implements a basic version of chess (GUI pending) that restricts the players to proper chess rules by tracking the pieces and checking input moves for validity.

Project design: This project was built using java and inheritance to create easy implementation of pieces. A GUI is included which runs and displays the game. Almost full functionality is involved with some minor bugs. When a piece is chosen, its possible moves are highlighted. Restrictions are included to ensure a proper and fair game of chess.

Implementation: An abstract piece class was used to create each type of piece. A board class was also created to keep track of the board as a whole. The board processes movements, captures, and user input. A GUI and backend exist to run the board. The backend prints the board to the terminal as a display while the GUI implements the same functionality, but in a more user friendly fashion.
